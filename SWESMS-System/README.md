# Smart Workshop Energy & Safety Monitoring System (SWESMS)

## Project Description
SWESMS is a C++ console application developed to monitor workshop equipment usage, electricity consumption, and safety status.

## Features
- Equipment record management
- Daily energy consumption calculation
- Electricity cost estimation
- Temperature safety monitoring
- Maintenance alerts

## Technologies Used
- C++
- Eclipse IDE
- GitHub

## Energy Formula
kWh = (Watts / 1000) × Hours Used

## Cost Calculation
Electricity Rate = RM0.50 per kWh

## Surcharge Rule
If total daily energy exceeds 8kWh → 20% surcharge applied.
